The PaCE Dashboard allows us to reveal relative trends between different topics driving the discourse regarding populism, nativism and civic engagement. We used Common Crawl, granting us the ability to scan web crawls from every year, 2013 to the present. The pipeline scanned thousands of keywords and search criteria; those keywords then got filtered through AI algorithms trained to recognize the different narrative topics.

![The PaCE Dashboard showing relative trends between populism-related topics](/uploads/2022/03/Screenshot-from-2022-03-20-15-56-26.png)

The public dashboard is no longer online, but the source code remains archived in the [PaCE keyword scanner repository](https://github.com/CitizensFoundation/pace-keyword-scanner).
