As a part of the [Icelandic Constitution Crowdsourcing](/impact/icelandic-constitution-crowdsourcing/) project, we launched an educational game to teach citizens in Iceland what constitutions are about from a high level. The game is built using a new open-source Citizens Foundation deep policy game engine called [Open Active Policy](https://github.com/CitizensFoundation/open-active-policy).

Citizens Foundation America's Josh Lanthier-Welch later wrote the detailed case file on the game, describing the goal as "an engaging fun experience" that still taught constitutional history and design. [Read Josh's case file about the creation of the game.](/blog/citizens-foundation-case-file-constitution-game/)

Open Active Policy is now a legacy platform, but the Make Your Constitution demo still works and remains a useful example of how game mechanics can make complex democratic design easier to understand.

[Play Make Your Constitution here.](https://demo-make-your-constitution.yrpri.org/)
